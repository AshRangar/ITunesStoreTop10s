<h5> Uses Apple's RSS Feeds <h5>
<h5> List of supported categories </h5>
<ul>

    <li>Free Apps</li>
    <li>Movies</li>
    <li>TV Seasons</li>
    <li>TV Episodes</li>
</ul>
